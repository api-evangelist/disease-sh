# disease.sh

APIs.json 0.19 provider profile for [disease.sh](https://disease.sh) — the Open Disease Data API.

disease.sh provides free, open, and unauthenticated access to global disease statistics including COVID-19 worldwide and per-country data, US state breakdowns, historical case counts, vaccine and therapeutic coverage, COVID-19 variant data, and CDC influenza surveillance data. The API has served over 100 billion requests and requires no signup or API key.

- **API Documentation:** https://disease.sh/docs/
- **OpenAPI Spec:** https://github.com/disease-sh/API/blob/master/public/apidocs/swagger_v3.json
- **GitHub Organization:** https://github.com/disease-sh
- **Base URL:** https://disease.sh/v3
- **Authentication:** None
- **Pricing:** Free
